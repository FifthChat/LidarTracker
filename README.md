# LidarTracker v3.0

A lidar tool made with Touchdesigner C++ CHOP, which can support HOKUYO, SLAMTEC, EAI and SIMINICS.

- Connect multiple lidars.
- Multiple lidars merge.
- Add trigger range
- TUIO & UDP

This version is only available in Touchdesigner 2022+.
